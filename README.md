# vue-photo-zoomer

基于vue2.x设计的图片放大镜组件。


## DEMO


## API


参数 | 说明 | 类型 | 默认值
--- | --- | --- | ---
 `url` | 图片的URL | String | -
 `urlFull` | 放大后的图片URL  | String | `url`
 `scale` | 放大倍数 | Number | 2
 `unit` | 计算单位 | String | `'px'`
 `magnifierStyle` | 放大镜的样式 | Object | -  
 `viewerStyle` | 观察区域的样式 | Object | -
